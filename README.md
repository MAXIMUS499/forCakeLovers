## Setup

### git clone

```bash
$ git clone https://github.com/MAXIMUS499/forCakeLovers.git
$ cd forCakeLovers/
$ npm install
```

## Gulp tasks

In order to follow changes in the project, run `gulp watch` 

```sh
$ gulp watch
```
